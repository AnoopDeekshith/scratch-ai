# Scratch.ai

AI-powered real-time lecture companion for college students. Get structured, AI-enhanced study notes as your professor speaks.

## Features
- ✅ Real-time transcription
- ✅ Slide upload (PDF/PPTX/DOCX)
- 🚧 AI note generation (Step 4)
- 🚧 Mermaid diagrams
- 🚧 Smart to-dos & quizzes

## Tech Stack
Next.js 14+ • TypeScript • Tailwind • Anthropic Claude • Supabase

## Quick Start
```bash
npm install
npm run dev
```

Add API keys to `.env.local` (see `.env.local` for template)
